How does javascript work?

JavaScript is a high-level programming language. Nowadays it's a popular programming language which is used to make website dynamic. it runs in browser through V8 enginee. Actually V8 engine is a "just in time compiler" which progrrammed by C++ . Some browser has it's own engine to run javascript for example Firefox has "Spider Monkey" , Edge-Browser has "Chakra" and Safari has "Javascript Cores".

What does javascript event loop do?

Javascript is also a single threaded and interpreted Asynchronus language . Actually It works in two ways when working on V8 Engine. these are Synchronous and Asynchronous methods. the V8 engine runs code line by line and adds function to call stack. setTimeout and setInterval are two event loop. Event Queue is a special queue, which keeps tracking of all functions queues, which are needed to be pushed into the call stack. The event queue is also responsible for sending new functions to the track for processing. JavaScript runs these code like a loop is called javascript event loop. This is what JavaScript Event Loop do.
